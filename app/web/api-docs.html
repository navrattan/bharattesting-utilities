BharatTesting API Documentation v1.0.0

Welcome to the official API for India-specific test data. This API is optimized for AI Agents, developers, and QA teams.

Authentication

Basic usage is anonymous and keyless. No signup required for up to 60 requests/hour.

Endpoints

GET /api/v1/aadhaar

Generate valid 12-digit Aadhaar numbers with Verhoeff checksum.

Params: count (1-100), format (json, csv, xml)

GET /api/v1/pan

Generate entity-aware PAN numbers.

Params: type (P: Individual, C: Company, T: Trust), count

GET /api/v1/gstin

Generate state-aware GSTIN numbers with Mod-36 checksum.

Params: stateCode (e.g., 27 for MH), count

GET /api/v1/profile

Generate complete Indian user profiles (Name, Phone, Email, Address).

Params: state, count

SDK & Code Snippets

cURL

curl "https://bharattesting.com/api/v1/aadhaar?count=5"

Node.js

const response = await fetch('https://bharattesting.com/api/v1/profile?count=1');
const data = await response.json();

Python

import requests
res = requests.get("https://bharattesting.com/api/v1/gstin?stateCode=27")
print(res.json())
