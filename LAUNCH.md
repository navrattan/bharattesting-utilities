## 📊 What's Built

### 🇮🇳 Indian Data Faker
✅ **9 Identifiers**: PAN, GSTIN, Aadhaar, CIN, TAN, IFSC, UPI, Udyam, PIN Code  
✅ **5 Templates**: Individual, Company, Proprietorship, Partnership, Trust  
✅ **Cross-field Consistency**: PAN↔GSTIN, state↔PIN, IFSC↔bank↔UPI  
✅ **Export Formats**: CSV, JSON, XLSX, Clipboard  
✅ **Bulk Generation**: Up to 10,000 records in <3 seconds  

### 📄 Document Scanner
✅ **Computer Vision**: Canny edge detection + contour approximation  
✅ **6 Filters**: Original, Auto-Color (CLAHE), Grayscale, B&W, Magic Color, Whiteboard  
✅ **OCR Integration**: Google ML Kit text recognition (on-device)  
✅ **Multi-page Support**: Batch scanning with reorder capability  
✅ **Searchable PDF**: Export with invisible text layer  
✅ **Web Fallback**: Upload image + manual crop for web users  

### 🖼️ Image Size Reducer  
✅ **Smart Compression**: Quality slider with real-time size estimation  
✅ **Format Support**: JPEG, PNG, WebP, AVIF (platform-dependent)  
✅ **Resize Presets**: Thumbnail, Small, HD, Full HD, 4K, Custom, Percentage  
✅ **Metadata Stripping**: Remove EXIF, GPS data for privacy  
✅ **Batch Processing**: Up to 50 images with ZIP export  
✅ **Before/After Preview**: Visual comparison slider  

### 📑 PDF Merger
✅ **Multi-file Support**: Merge up to 20 PDFs (100MB total)  
✅ **Visual Interface**: Drag & drop page reordering with thumbnails  
✅ **Page Operations**: Rotate (90°/180°/270°), delete unwanted pages  
✅ **Security**: Optional password protection  
✅ **Auto-bookmarks**: Generated from source filenames  
✅ **Progress Tracking**: Real-time merge progress indicator  

### 🔧 String-to-JSON Converter
✅ **Format Auto-detection**: Raw string, CSV, YAML, XML, URL-encoded, INI  
✅ **6 Auto-repair Rules**: Trailing commas, single quotes, unquoted keys, comments, Python literals  
✅ **Syntax Highlighting**: Beautiful JSON output with flutter_highlight  
✅ **Error Reporting**: Line/column precision with suggested fixes  
✅ **Export Options**: Copy, Download, Minify, Pretty-print toggle
